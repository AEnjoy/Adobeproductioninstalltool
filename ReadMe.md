# What's this?

这个版本库收录了Adobe Photoshop 和 Premiere Pro的绿色版安装包,资源来自[zd423](https://www.423down.com/).

除此之外,该版本库还包含了在线安装的相关配置文件

这个版本库还将作为InstallTool的源代码库存在,并长期更新.



项目使用宽松MIT协议分发,鼓励用于学习等用途

由于本人拙劣的代码水平,各位大神看官笑笑就好=.=
